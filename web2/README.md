# Практична робота №2: JavaScript. Динамічний контент

**Варіант 2:** Вітрова електростанція (Моніторинг параметрів у режимі реального часу: швидкість вітру, обертання ротора, вироблена потужність, кут нахилу лопатей).

## Про проєкт
Це інтерактивний веб-дашборд, створений для моніторингу стану вітрової електростанції. Проєкт виконано згідно з вимогами до практичної роботи №2, з фокусом на використання JavaScript для маніпуляцій з DOM та оновлення даних без перезавантаження сторінки.

**Основний функціонал:**
- Динамічне відображення поточних параметрів генератора.
- Ручне (за допомогою кнопки) та автоматичне (таймер) оновлення даних.
- Візуальна індикація стану показників (Норма, Увага, Критично) зі зміною кольорів.
- Інтерактивний графік зміни потужності у часі.
- Візуалізація напрямку вітру (компас) та спрацювання гальмівної системи при критичній швидкості вітру (>20 м/с).

## Використані технології
- **HTML5:** семантична структура дашборду, атрибути доступності (ARIA).
- **CSS3:** сучасний дизайн з ефектом Glassmorphism (матове скло), CSS Grid для розташування карток, анімації критичних станів та адаптивність для мобільних пристроїв.
- **JavaScript (ES6+):** генерація випадкових даних у заданих діапазонах, робота з DOM-елементами, налаштування таймерів (`setInterval`), логіка обробки подій.
- **Chart.js:** зовнішня бібліотека для побудови графіка у реальному часі.
