# Практична робота №1: Розробка інформаційного веб-сайту енергетичного об’єкта

**Варіант 2:** Сонячна електростанція «Сонячний Степ» (Потужність: 50 МВт, Напруга: 0,4/35 кВ, Місцезнаходження: Херсонська обл., с. Калинівка).

## Про проєкт
Це адаптивний односторінковий веб-сайт, створений для представлення енергетичного об'єкта. Проєкт виконано згідно з вимогами до практичної роботи №1. 

Сайт складається з таких розділів:
- Головний екран 
- Навігаційне меню (згортається на мобільних пристроях)
- Детальний опис станції
- Таблиця з технічними параметрами
- Схема та фотографії об'єкта
- Контактна інформація 

## Використані технології
- **HTML5:** для створення семантичної структури.
- **CSS3:** для власної стилізації (кольорова схема, hover-ефекти, анімації).
- **Bootstrap 5:** для швидкого створення адаптивної сітки та компонентів.

## Як відкрити проєкт
1. Завантажте код з цього репозиторію.
2. Просто відкрийте файл `index.html` у будь-якому браузері (подвійним кліком). 
*Або можна використати розширення Live Server у VS Code для зручного перегляду.*
