# Практична робота №3: HTML-форми та серверна обробка (Node.js + Express)

**Варіант 2:** Облік сонячних електростанцій .

## Про проєкт
Це веб-додаток для реєстрації та обліку приватних сонячних електростанцій, створений з використанням клієнт-серверної архітектури. Проєкт виконано згідно з вимогами до практичної роботи №3, з фокусом на створення HTML-форм, обробку даних на сервері за допомогою Node.js та Express, а також збереження інформації у форматі JSON.

**Основний функціонал:**
- Збір даних про СЕС через інтерактивну форму з валідацією полів.
- Логіка умовних полів (показ поля ємності акумуляторів лише за їх наявності).
- Автоматичний розрахунок орієнтовної річної генерації на основі введеної потужності.
- Серверне API (GET, POST, DELETE) для керування записами.
- Збереження даних на сервері у файлі `spp.json`.
- Динамічне оновлення загальної статистики мережі (кількість СЕС, сумарна потужність та ємність) та списку зареєстрованих об'єктів без перезавантаження сторінки.

## Використані технології
- **HTML5:** семантична розмітка форми, використання типів `input` та атрибутів валідації (`required`, `min`).
- **CSS3:** сучасний адаптивний дизайн, Grid-розмітка для карток статистики.
- **JavaScript (ES6+):** клієнтська логіка (Fetch API для асинхронних запитів, обробка подій форми, оновлення DOM).
- **Node.js:** середовище виконання серверного коду.
- **Express.js:** веб-фреймворк для налаштування маршрутизації та обробки JSON/URL-encoded даних.
